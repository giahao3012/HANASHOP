# HanaShop
Java Web - Food and Drink Web

-Techs: Back-end Java, log in with Google API</br>
-Front-end: free bootstrap template</br>
-Server: Tomcat 8.x</br>
-Pattern: MVC</br>
-Database: Microsoft SQL server</br>
